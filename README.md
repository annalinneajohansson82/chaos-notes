# ChaosNotes

This project was generated using [Angular CLI](https://github.com/angular/angular-cli) version 21.2.11.

## Prerequisites

Before you start, make sure you have the following installed:

1. **Node.js** (v20.19 or v22.12 or later) — download from [nodejs.org](https://nodejs.org/)
   - Verify it is installed by running: `node -v`
2. **npm** (comes bundled with Node.js)
   - Verify it is installed by running: `npm -v`
3. **Angular CLI** — install globally by running:
   ```bash
   npm install -g @angular/cli
   ```
   - Verify it is installed by running: `ng version`

## Getting started

### 1. Install dependencies

From the project root directory, install all required packages:

```bash
npm install
```

This only needs to be done once, or again after pulling changes that update `package.json`.

### 2. Start the development server

```bash
npm start
```

### 3. Open the app

Once the server is running, open your browser and navigate to:

```
http://localhost:4200/
```

The app will automatically reload whenever you save changes to a source file.
